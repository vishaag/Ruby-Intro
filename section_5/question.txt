Instructions
Implement a Ruby class ‘Stack’ that encapsulates a stack. The initializer must take an integer argument that defines the size of the stack. Implement the following methods

push - Pushes an element to the top the stack. Returns nil if the stack is already full

top - Returns the top element of the stack. Returns nil if the stack is empty

pop - Returns and removes the top element of the stack. Returns nil if the stack is empty

size - Returns the size of the stack
